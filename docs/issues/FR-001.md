# [FR-001] Endless Core Loop & Combat

## Labels
`MUS`, `core-mechanics`, `combat`

## User Story
As a player, I want to shoot enemies, collect power-ups, and survive endlessly so that I can achieve my highest score in a pure skill-based environment.

## Proposed Solution

### Overview
Implement a continuous game loop using `requestAnimationFrame`. The game will infinitely spawn waves of enemies with increasing frequency or difficulty multipliers. The player controls a ship (`PlayerObject`) that can move, shoot, and collect drops (missiles, bombs, extra lives).

### Implementation Flow
1. Set up the Canvas rendering context.
2. Build the `PlayerObject` with movement physics (friction, acceleration).
3. Create `EnemyObj` classes with basic movement (linear, hover, wave).
4. Implement AABB or simple circular collision detection (`Tabrakan`).
5. Track `game.score` and scale difficulty based on score/time.

### Technical Approach
```javascript
// Example collision check pattern
if (Tabrakan(player.getHitbox(), enemy.getHitbox())) {
  handlePlayerHit();
}
```

## Acceptance Criteria
- [x] Player can move using WASD/Arrows or Mouse.
- [x] Player can fire standard lasers.
- [x] Enemies spawn off-screen and travel towards the player.
- [x] Collisions correctly register damage and destroy entities.
- [x] Score increases upon enemy defeat.
