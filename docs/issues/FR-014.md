# [FR-014] Foreground Micro-Debris

## Labels
`MUS`, `enhancement`, `visuals`

## User Story
As a player, I want to see space dust flying past the foreground so that my brain subconsciously registers the intense speed of the ship.

## Proposed Solution

### Overview
A highly-transparent particle system rendered *above* the player to provide extreme depth cues.

### Implementation Flow
1. Create a `spaceDustArray` pooled object system.
2. Render tiny white/blue lines scaling their length based on velocity.
3. Draw them at low opacity (0.1 - 0.2) after everything else.

### Technical Approach
```javascript
// Draw dust as streaks
ctx.globalAlpha = 0.15;
ctx.beginPath();
ctx.moveTo(dust.x, dust.y);
ctx.lineTo(dust.x, dust.y - dust.velocity * 2);
ctx.stroke();
```

### Key Considerations
- Must be nearly invisible during normal play to prevent visual clutter.
- Only scales up visibly during Adrenaline Surge.

## Acceptance Criteria
- [ ] Dust particles fly past at high speed.
- [ ] Dust does not interfere with reading enemy bullets.
