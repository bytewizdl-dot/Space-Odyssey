# [FR-010] Deep-Space Parallax

## Labels
`MUS`, `enhancement`, `visuals`

## User Story
As a player, I want the background planets and stars to scroll at different speeds so that I feel a realistic sense of depth and scale.

## Proposed Solution

### Overview
Separate the existing background into multiple layers that scroll independently to create a 3D parallax effect.

### Implementation Flow
1. Load multiple background layers (e.g., distant stars, planets, close stars).
2. Apply different scrolling multipliers to each layer in `updateGame()`.
3. Draw them in order (back to front) in `drawGame()`.

### Technical Approach
```javascript
// Layer 1: distant
bg1.y += globalSpeed * 0.2;
// Layer 2: planets
bg2.y += globalSpeed * 0.5;
```

### Key Considerations
- Layer rendering order is critical.
- Keep image assets lightweight to avoid memory bloat.

## Acceptance Criteria
- [ ] At least 3 distinct depth layers.
- [ ] Backgrounds loop seamlessly.
