# [FR-004] Audio & Crossfade Pipeline

## Labels
`MUS`, `audio`, `systems`

## User Story
As a player, I want seamless music transitions and non-overlapping sound effects so that the audio experience is professional and highly polished.

## Proposed Solution

### Overview
Implement an `AudioMixer` class to handle SFX pooling and throttling, preventing audio context crashes when hundreds of bullets fire. Implement BGM crossfading in `AudioManager.js`.

### Implementation Flow
1. Create `AudioMixer` with voice limits (e.g., max 10 laser sounds at once).
2. Create `playSfxWithVolume()` wrapper.
3. Build logic for smoothly fading out the Main Menu BGM over 2 seconds when starting the game.
4. Build crossfade logic for transitioning to Boss/Surge music.

### Technical Approach
```javascript
class AudioMixer {
  static play(src, volume) {
    // Check if voice limit reached
    // Play from pool
  }
}
```

## Acceptance Criteria
- [x] Firing lasers rapidly does not cause audio distortion or dropping.
- [x] Transitioning from Menu to Game fades the music gracefully (no abrupt cuts).
- [x] Adrenaline Surge cleanly interrupts and replaces the current BGM.
