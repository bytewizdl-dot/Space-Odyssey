# [FR-005] Immersive UI & Menus

## Labels
`MUS`, `ui`, `frontend`

## User Story
As a player, I want a 3D-angled menu and polished HUD so that the game feels like a premium indie experience right from the start.

## Proposed Solution

### Overview
Use HTML and CSS to create a perspective-transformed main menu floating over the Canvas. The HUD will also be drawn using Canvas `fillText` and `drawImage` in screen-space.

### Implementation Flow
1. Build `Main.html` with a `#mainMenu` container.
2. Apply CSS `transform: perspective(1000px) rotateY(-15deg)` for 3D effect.
3. Add animated glow, hover effects, and staggered entry animations (`animation-delay`).
4. Design the in-game HUD (Lives as heart icons, Bombs, Missiles, Score).

### Technical Approach
```css
.menu-content-angled {
  transform: perspective(1000px) rotateY(-15deg) translateZ(0);
  transform-style: preserve-3d;
}
```

## Acceptance Criteria
- [x] Main menu is visually striking and angled in 3D space.
- [x] Buttons have clear hover states and click sounds.
- [x] HUD accurately reflects player state (`game.score`, `abilityCharges`).
- [x] UI overlays (like the Fullscreen button) scale properly on different monitors.
