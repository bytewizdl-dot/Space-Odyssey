# [FR-006] Advanced AI Spawns

## Labels
`Future`, `gameplay`, `ai`

## User Story
As a player, I want enemies to spawn in complex formations and spline paths so that gameplay remains fresh and unpredictable over long sessions.

## Proposed Solution

### Overview
Upgrade enemy spawning to use mathematical formation patterns (Sine, Cosine, Lissajous curves). This creates the illusion of complex "AI" intelligence using virtually zero CPU cycles, as it relies on deterministic math instead of pathfinding or flocking physics.

### Implementation Flow
1. Create a `FormationMath` utility to generate X/Y offsets based on `game.frames`.
2. Update `EnemyObj` to support "Pattern Modes" (Wave, Circle, Dive).
3. Implement "Follower" logic where enemies in a squadron simply lag behind the leader's pattern by a few frames.

### Technical Approach
```javascript
// Performance-First: Use sine/cosine offsets instead of pathfinding
this.x -= speed;
this.y += Math.sin(this.clock * frequency) * amplitude;
```

### Key Considerations
- **Mathematical Patterns**: Use `Math.sin` and `Math.cos` to create curves; it's virtually free for the CPU.
- **Batch Processing**: Avoid complex logic inside the `update` loop.
- **Formation Offsets**: Offset followers by a fixed time `t` rather than calculating distances.
