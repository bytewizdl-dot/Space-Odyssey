# [FR-016] Radial Edge-Stretching

## Labels
`Future`, `enhancement`, `visuals`

## User Story
As a player, I want stars at the edges of the screen to warp into lines during Adrenaline Surges so that I feel the intensity of light-speed travel.

## Proposed Solution

### Overview
A "Warp Tunnel" effect that isolates motion blur strictly to the peripheral vision to avoid nausea.

### Implementation Flow
1. Track stars in a dedicated `warpLinesArray`.
2. When drawing, calculate the star's distance from the center X axis.
3. If the star is far left or far right, elongate its Y-scale and apply color dodge.

### Technical Approach
```javascript
let distFromCenter = Math.abs(star.x - canvasWidth/2);
if (distFromCenter > 200) {
  let stretchFactor = (distFromCenter - 200) / 100;
  // Draw elongated streak
}
```

### Key Considerations
- Strict CPU limits required (max 40 warp lines).
- Must only activate during max speed Adrenaline Surge.

## Acceptance Criteria
- [ ] Edge stars stretch into lines during Surge.
- [ ] Center stars remain dotted to keep gameplay readable.
