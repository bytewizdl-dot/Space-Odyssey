# [FR-009] Biological Emissives

## Labels
`MUS`, `enhancement`, `visuals`

## User Story
As a player, I want to see enemy engines and cores naturally pulse so that they feel like living, organic threats.

## Proposed Solution

### Overview
Use standard 2D Canvas alpha manipulation and additive blending to create a "heartbeat" effect on enemy sprites without needing complex shaders.

### Implementation Flow
1. Identify the draw cycle for Enemy objects.
2. Calculate a pulsing alpha value using `Math.sin(Date.now() / 150)`.
3. Draw an emissive mask or simply pulse the brightness using `ctx.globalCompositeOperation = 'lighter'`.

### Technical Approach
```javascript
let pulse = (Math.sin(Date.now() / 150) + 1) / 2; // 0 to 1
ctx.globalAlpha = 0.5 + (pulse * 0.5);
ctx.globalCompositeOperation = 'lighter';
// draw emissive sprite
```

### Key Considerations
- Ensure the effect doesn't break normal hitboxes.
- Performance is O(1) per enemy, practically free.

## Acceptance Criteria
- [ ] Enemies visually pulse over time.
- [ ] The pulse feels organic (sine wave based).
