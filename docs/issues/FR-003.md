# [FR-003] "Adrenaline Surge" System

## Labels
`MUS`, `gameplay-event`, `chaos`

## User Story
As a player, I want periods of extreme chaos and difficulty spikes so that the pacing remains tense, unpredictable, and exciting.

## Proposed Solution

### Overview
A scheduled event system that temporarily plunges the game into "Surge Mode". During a Surge, the background turns red, enemies become hyper-aggressive, unique mini-bosses spawn, and the camera shakes constantly.

### Implementation Flow
1. Add a `surgeSchedulerTimer` to the main game loop.
2. Phase 1: Warning (Sirens, red UI flashing, BGM cuts out).
3. Phase 2: Active Surge (Heavy metal BGM, violent screen shake, plasma bullets).
4. Phase 3: Cooldown (Return to normal BGM, clear remaining enemies).

### Technical Approach
```javascript
if (game.surgePhase === 1) {
    // Spawn tougher enemies
    shootChance *= 2.5;
    // Apply violent shake
    surgeShake.update();
}
```

## Acceptance Criteria
- [x] Surge event triggers reliably based on a timer or score threshold.
- [x] Audio transitions cleanly into Surge music.
- [x] Visual indicators (red flashing, warnings) alert the player.
- [x] Game returns to normal pacing after the event concludes.
