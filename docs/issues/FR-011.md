# [FR-011] Faux Heat Distortion (Lightweight)

## Labels
`MUS`, `enhancement`, `visuals`

## User Story
As a player, I want to see shockwaves and engine trails blend dynamically so that explosions feel violent and powerful.

## Proposed Solution

### Overview
To avoid the CPU death of `getImageData`, we fake distortion using overlapping, highly transparent radial gradients with `overlay` or `color-dodge` blending.

### Implementation Flow
1. Create a `createShockwave()` function.
2. Draw an expanding radial gradient that shifts from transparent white to fully transparent.
3. Use a strong composite mode to bend the colors beneath it visually.

### Technical Approach
```javascript
ctx.globalCompositeOperation = 'overlay';
let grad = ctx.createRadialGradient(x, y, 0, x, y, radius);
grad.addColorStop(0, 'rgba(255,255,255,0.8)');
grad.addColorStop(1, 'rgba(255,255,255,0)');
```

### Key Considerations
- Must NOT use `getImageData`.
- Only spawn for massive events (boss death, bombs).

## Acceptance Criteria
- [ ] Explosions have an expanding shockwave effect.
- [ ] Framerate remains 120fps.
