# [FR-012] Relative Kinetic Drag

## Labels
`MUS`, `enhancement`, `physics`

## User Story
As a player, I want the enemies to physically react to my ship's acceleration so that the "treadmill effect" is broken and the world feels grounded.

## Proposed Solution

### Overview
Apply a downward velocity modifier to all physical entities (enemies, particles) relative to the background's current acceleration speed.

### Implementation Flow
1. Calculate the background speed multiplier (Surge speed vs Normal speed).
2. During `Enemy.update()`, apply `this.y += (bgSpeed - normalSpeed) * 0.8`.
3. Enemies will look like they are fighting against massive wind resistance.

### Technical Approach
```javascript
let surgeDrag = Math.max(0, gameSettings.currentScrollSpeed - gameSettings.baseScrollSpeed);
this.y += surgeDrag * 0.5; // Drag factor
```

### Key Considerations
- Fast enemies should still be able to move forward, just slower relative to the screen.

## Acceptance Criteria
- [ ] Enemies visually slide down the screen when Adrenaline Surge triggers.
- [ ] The relative speed logic connects the background to the foreground.
