# [FR-008] Enhanced Visuals & Shaders

## Labels
`MUS`, `visuals`

## User Story
As a player, I want advanced lighting and bloom effects so that the game looks visually stunning.

## Proposed Solution
Implement additive blending (`lighter`) for projectiles and glow sprites.

## Acceptance Criteria
- [x] Bullets have radial glowing bloom.
- [x] High-performance glow rendering using pre-rendered offscreen canvases.
